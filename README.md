**Projeto de estagiários da Teknisa. 

O projeto trata-se de fazer um CRUD de desenvolvedores em Node.js;

**Inicie o projeto baixando o Node https://nodejs.org/en/;
-> Após a instalação, execute o comando no cmd para certificar que foi instalado corretamente;
(node -v) e (npm -v)

**Criamos o package.json usando o comando (npm init);

**Instalamos o pacote "express" com o comando (npm install express);

**Outros pacotes necessários foram "sequelize sqlite 3" e "body-parse";
-> Fizemos a instalação com os seguintes comandos (npm install sequelize sqlite3) e (npm install body-parser);
